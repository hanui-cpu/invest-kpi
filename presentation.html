배경 · WHY
투자전략팀의 역할 —
KPI를 실시간으로 한눈에
투자전략팀은 투자실 전체의 성과를 데이터로 정리하고, 의사결정을 지원하는 팀입니다.
그런데 핵심 KPI 수치를 꺼내는 일이 생각보다 힘들었습니다.
①
구글 시트에 묻혀 있는 숫자들
수백 행의 투자 데이터 — 연도별·분기별 실적이 한눈에 보이지 않음
②
보고할 때마다 수작업 정리
투자목표금액, 회수목표금액 진척도를 매번 수식으로 계산하고 차트를 새로 그림
③
개발 리소스 없음
외부 의뢰 시 수백만 원 + 수개월 — 현실적으로 불가능
투자전략팀이 원한 것
  • 투자팀 KPI — 도메인 집중도, 극초기 투자 달성률
  • 올해 투자목표금액 대비 실시간 진척도
  • 회수목표금액 5개년 모니터링
  • 보고 때 클릭 한 번으로 PDF/PPT 출력

Claude 활용 방식
실제로 어떻게 대화했나
1
요구사항을 말로 설명
"구글 시트 데이터를 읽어서 연도별·분기별로 필터링되는 차트를 만들어줘"
2
오류가 나면 그대로 붙여넣기
에러 메시지를 복사해서 "이거 왜 안 되는 거야?" → 원인 분석 + 즉시 수정
3
디자인도 말로 요청
"카드 순서를 집중도 → 극초기 → 회수 → 투자한도 순으로 바꿔줘"
4
작업 이력도 자동 생성
매 작업 후 "오늘 변경사항 kpi.md에 정리해줘" → 문서화 자동 완성
핵심 깨달음
Claude는 코드를 모르는 저를 대신해서
"의도"를 이해하고 구현해줬습니다.
실수하면 이유를 설명해주고, 더 나은 방법도 제안했어요.
Claude가 해결한 실제 문제들
  • 구글 시트 컬럼 매핑 오류 전면 수정
  • 의결일자 기준 변경 시 회수 0건 버그 OR 조건으로 픽스
  • PDF/PPT — 아코디언 자동 펼침 후 캡처·복원
  • 도메인 호버 툴팁, 화면 끝 자동 방향 전환
